## ✨ Features

- One-liner install (`curl | bash`)
- Supports both **user** (`~/.local/bin`) and **system** (`/usr/local/bin`) installation
- Automatically installs SDKMAN! + pinned **Java 21 (Temurin)** + **Maven 3.9.14**
- Creates (or safely re-uses) a minimal Spring Boot 3.3.5 "Hello World" project
- **Project preservation by default** — re-runs keep your existing files (`pom.xml`, sources, properties)
- Full self-management: `self-update`, `version-check`, `self-uninstall`
- Rich flag support: `--force`/`--reset`, `--project-dir`, `--no-run`, `--quiet`, `--json`
- Multi-shell PATH setup (bash, zsh, fish)
- Extremely defensive "CIAO" coding style with repeated safe defaults and anti-simplification guards
- Full single-source-of-truth output system (`output_text` / `output_json`)
- Smart help text with automatic EOL warning for legacy versions

## 🚀 Quick Installation

**For normal users:**
```bash
curl -fsSL https://raw.githubusercontent.com/Wilgat/springboot3/main/springboot3 | bash
```

**System-wide (requires sudo):**
```bash
curl -fsSL https://raw.githubusercontent.com/Wilgat/springboot3/main/springboot3 | sudo bash
```

After installation, simply run:
```bash
springboot3
```

The application will start at **http://localhost:8080**

## 📖 Usage

```bash
springboot3                    # Setup environment + build + run (preserves project)
springboot3 --reset            # Full reset: delete and regenerate project files
springboot3 --no-run           # Setup only (no build/run) — useful for CI/Docker
springboot3 --project-dir <path>   # Run against an existing project
springboot3 version            # Show current version
springboot3 version-check      # Compare with latest on GitHub
springboot3 self-update        # Update to latest version
springboot3 self-uninstall     # Remove from the system
springboot3 help               # Show this help
```

### Key Behaviors
- **Normal run**: Preserves your existing project folder, `pom.xml`, Java source, and `application.properties`.
- **`--reset`** / **`--force`**: Completely wipes and regenerates the project for a clean slate.

## Important Platform Notes

### Alpine Linux
Requires **bash** (SDKMAN! does not work reliably under BusyBox ash).  
The script auto-detects Alpine and gives clear instructions:
```bash
apk add bash
bash <(curl -fsSL https://raw.githubusercontent.com/Wilgat/springboot3/main/springboot3)
```

### Other Platforms
- **macOS**, **Git Bash (Windows)**, **Ubuntu/Debian**, **RHEL/Rocky** — all supported with defensive fallbacks.

## Project Philosophy

`springboot3` is part of the deliberate **springboot2 / springboot3 / springboot4** family.  
Each tool is maintained separately with its own **pinned** Spring Boot version to provide stable, reproducible environments for different use cases:

- `springboot2` → Legacy / long-term support
- `springboot3` → Current stable (this project)
- `springboot4` → Bleeding edge

**Spring Boot 3.3.5** and **Java 21 (Temurin)** are **intentionally pinned**. We are aware of newer versions, but this script exists specifically for users who need or prefer 3.3.5.

The script follows the strict **CIAO** defensive coding philosophy (Caution • Intentionality • Anti-fragility • Ownership) with heavy comments and `!!! DO NOT MODIFY OR SIMPLIFY !!!` blocks to survive harsh environments and protect against accidental simplification.
